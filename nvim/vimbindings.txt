Vim shortcuts:


Window management:
ctrl + w + movement (HJKL)
leader + e : toggle file tree

Finding files:
leader + ff : find files
leader + fs : find string
leader + fc : find string under cursor
leader + fb : show open buffers

harpoon:
leader + a : add file
ctrl + (jkl;) : jump to file
ctrl + e : show all files


issues:
leader + xx : show issues


Terminal:

leader + tc : open temp terminal

Autocomplete:
ctrl + space : complete
ctrl + e :close

Refactor:

leader + rn : rename
leader + ca : code action
K : show doc (cursor)
leader + rf: rename file
leader + oi : orignise imports
leader + ru: remove unused vars
